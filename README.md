# Proyecto-SraMaria
🧾 Sistema de Registro de Ventas
📌 Acerca del proyecto

Aplicación para registrar ventas y gestionar pagos en una tienda de abarrotes.
Permite llevar control básico de ingresos y de clientes con deudas pendientes.

🚀 Funcionalidades
Registrar ventas con producto, cantidad y precio
Definir tipo de pago: contado, parcial o fiado
Guardar clientes con deudas pendientes
Registrar pagos realizados por clientes
Consultar deudores y montos adeudados
Visualizar ingresos del día
Buscar ventas por fecha o cliente
👤 Usuarios
Vendedor: registra ventas
Propietaria: revisa ingresos
Cobrador: gestiona pagos de deudas
